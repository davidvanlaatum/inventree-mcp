# Operator Recipes

This file will contain first-release operator workflows once implementation begins.

Required first-release recipes are tracked in [PLAN.md](PLAN.md), including:

- ChatGPT connector OAuth setup.
- STDIO setup.
- Reverse-proxy HTTP deployment.
- Add or update a purchasable part.
- Reuse existing parameters.
- Add supplier and manufacturer links.
- Create initial stock.
- Upload or link an attachment.
- Set or replace a primary part image.
- Preview purchase order lines.
- Resolve structured clarification prompts.
